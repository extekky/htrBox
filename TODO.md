Три уровня пирамиды:
tokens.ts ← атомарные значения (gap, size, color-role)
↓ compose
components.ts ← собирает токены в именованные слоты компонента
↓ use
FormInput.tsx ← только логика, никаких сырых классов
Шаг 2 — переписать components.ts так, чтобы он составлял из токенов:
tsimport { focus, spacing, typography } from "@/styles/tokens";

Теперь если дизайнер скажет "сделай все мелкие отступы чуть больше" — меняешь spacing.inlineGapSm в одном месте, и это распространяется везде автоматически.

Когда будешь распространять систему на весь проект — первый шаг именно такой: аудит всех сырых gap-_ классов и замена их на spacing._ токены.
